# Code of Conduct

This project uses the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating, you are expected to keep discussions respectful, constructive,
and focused on improving the project. Unacceptable behavior includes harassment,
personal attacks, sustained disruption, and publishing another person's private
information.

Project maintainers may remove comments, close issues, or block contributors who
do not follow these expectations.

To report a conduct concern, open a private security advisory or contact the
maintainer through the repository owner profile. Do not include sensitive personal
information in a public issue.
