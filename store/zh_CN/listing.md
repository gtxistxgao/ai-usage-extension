## Name

AI Usage Tracker：Claude 与 Codex 限额

## Short Description

在弹窗、徽章和页面浮层中跟踪 Claude 与 ChatGPT (Codex) 的会话和每周限额。注重隐私，无需设置。

## Full Description

AI Usage Tracker：Claude 与 Codex 限额是一款浏览器扩展，适合希望获得更清晰、更可预测 AI 助手体验的用户。你可以跟踪消息限额，查看实时倒计时，并直接在服务页面上显示不打扰的使用量仪表，这样就能专注工作，而不会突然撞上限额。

无论你使用 Claude 和 ChatGPT 来编程、写作、研究还是深入解决问题，AI Usage Tracker 都能减少遇到限额阻断时的摩擦，并让你直接控制容量的监控方式。你可以选择浮层出现的位置，也可以选择如何跟踪自己的限额。

主要功能：

- 实时跟踪 Claude (claude.ai) 和 ChatGPT/Codex (chatgpt.com) 的限额
- 工具栏徽章一眼显示当前最高使用百分比
- 页面内上下文浮层，直接放在聊天输入框旁边
- 使用 Shadow DOM 的可折叠胶囊小组件，避免与宿主页面样式冲突
- 基于会话 (5 小时) 和每周 (7 天) 的容量监控
- 实时倒计时，准确显示限额何时重置
- 按需刷新按钮，以及每 5 分钟自动后台同步
- 100% 私密：直接从你自己的已认证浏览器会话中读取

用户选择 AI Usage Tracker 的原因：
- 快速、自动化的使用情况轮询，无需手动刷新页面
- 更清爽的 UI，可无缝融入 AI 聊天界面
- 更好地安排高强度编程、学习或写作会话的节奏
- 安全设计，无外部服务器、无跟踪、无遥测
- 简单的侧边面板控制，可快速启用或禁用

适合：
- 高频使用 Claude Code 或 ChatGPT 限额的开发者和高级用户
- 希望清晰了解配额的 Plus、Pro 和 Team 计划订阅者
- 希望避免深度工作流被打断的专业人士和学生
- 厌倦了毫无预警就看到“you've reached your limit”的任何人
