# Chrome Web Store Listing - Japanese

## Name

AI Usage Tracker: Claude と Codex の上限

## Short Description

Claude と ChatGPT (Codex) のセッション・週間上限をポップアップ、バッジ、オーバーレイで確認。プライバシー重視、設定不要。

## Full Description

AI Usage Tracker: Claude と Codex の上限は、より明確で予測しやすい AI アシスタント体験を求める人のためのブラウザ拡張機能です。メッセージ上限を追跡し、リアルタイムのカウントダウンを確認し、プロバイダーページ上に控えめな使用量メーターを直接表示できるため、突然の上限にぶつかることなく作業に集中できます。

Claude と ChatGPT をコーディング、執筆、調査、深い問題解決に使っている場合でも、AI Usage Tracker は上限ブロックに伴うストレスを減らし、容量の監視方法を直接コントロールできるようにします。オーバーレイをどこに表示するか、どのように上限を追跡するかを自分で選べます。

主な機能:

- Claude (claude.ai) と ChatGPT/Codex (chatgpt.com) のリアルタイム上限追跡
- 現在もっとも高い使用率を一目で示すツールバーバッジ
- チャット入力欄のすぐ横に配置されるコンテキスト対応のページ内オーバーレイ
- ホストページのスタイル衝突を防ぐ Shadow DOM 内の折りたたみ式カプセルウィジェット
- セッションベース (5時間) と週間 (7日間) の容量監視
- 上限がいつリセットされるかを正確に示すライブカウントダウン
- オンデマンド更新ボタンと 5 分ごとの自動バックグラウンド同期
- 100% プライベート: 自分の認証済みブラウザセッションから直接読み取ります

ユーザーが AI Usage Tracker を選ぶ理由:
- ページを手動で再読み込みせずに高速で自動的な使用状況ポーリング
- AI チャットインターフェイスに自然に統合される、よりクリーンな UI
- 集中的なコーディング、学習、執筆セッションのペース管理がしやすい
- 外部サーバー、トラッキング、テレメトリなしの安全な設計
- すばやく有効化・無効化できるシンプルなサイドパネル操作

こんな人に最適:
- Claude Code や ChatGPT の上限を頻繁に使う開発者とパワーユーザー
- クォータを明確に把握したい Plus、Pro、Team プランの利用者
- 深い作業の流れを中断したくないプロフェッショナルや学生
- 事前の警告なしに「上限に達しました」と表示されることに疲れた人
